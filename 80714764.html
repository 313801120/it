  • IT行业工资虚高? 程序员到底都干了什么?

  • 作者： 发表于：2018-09-16 20:50:45
    互联网行业在以大家想象不到的速度改变着这个世界，IT行业也成了众多年轻人选择的热门行业之一，就因为一句“IT行业高薪职业”，小编在知乎上看到这么一个疑问句“ IT 行业的工资是否虚高？”，引发了不同的声音，有些人说“这些人根本不知道程序员到底干的什么！”，也有人说“写个代码月薪1W+，还上到2W+？Are you kidding me......”
    　　
    　　小编看了很多的答案，就一个答案，小编个人觉得是比较中肯的回答，也是比较合理的回答，因为这个问题没有绝对的对与错，只能说是合理的。
    　　
    　　程序员到底做的是什么工作？
    　　如果你在一个互联网公司，你可能会这样调侃你的程序员同学：
    　　你工作不饱和啊，bug解完了吗?
    　　下面的对话也是常有的：
    　　A：在干嘛呢?
    　　B：苦逼加班解bug呢。
    　　你也许看出来了，大多数程序员常常受限于解bug的恶性循环中――以前版本的bug还没解完，新功能上线又出现一堆bug。解bug是一件特别阻碍生产力的事情，它是一件副产品，理论上员工应该把大部分精力都用在开发上，而大多数公司却在维护软件的稳定性上付出了巨大的成本。
    　　开发过程中产品的开发时间很重要，希望尽可能的完成开发并发布产品，例如希望一个月完成，但却忽略了产品发布后，可能需要十个月甚至更多的时间来维护它，这真的是一个本末倒置的怪圈。如果每个人都有意识提升自己的工作效率，提升自己的出品质量，把更多时间投入到有价值的事情上，公司的业绩不会差，公司赚钱了，你的工资也不会低。你可能会问如何才能提升工作效率，其实也很简单，做每件事之后，都要反思：
    　　还有更简单的方法吗?
    　　我该如何才能减少bug的数量呢?
    　　做这件事有通用的解决方案吗?
    　　这就是一个程序员的工作。
    　　
    　　IT行业哪个岗位最吃香？
    　　外媒网站Randstad US收集了目前IT领域各行业的数据，分析了劳动力趋势，包括合格候选人和可用职位的数量，市场增长幅度，薪资范围和雇主对具体技能的需求等，最终预测2017-2018年IT行业最好的5个职位：
    　　1. 大数据工程师 Big Data Engineer
    　　要成为大数据工程师，必备的技能有Java，Spark，Hadoop，Hive和Big Data。近两年伴随着大数据越来越火热，人工智能、深度学习更是占据着各大媒体头条，所以，有这些基础的程序员今年可以努力向大数据工程师的方向进攻!
    　　2.全栈工程师 Full Stack Developer
    　　一名合格的全栈工程师应具备这些技能――JavaScript，完整堆栈框架，Java，级联样式表和超文本标记语言。
    　　市场对全栈工程师的需求相较去年上升了117%，这个数据是非常惊人的!该行业对程序员的要求比较高。
    　　而在职的开发人员中，也有大量的公司正努力培养内部开发人员向全栈开发方向转变。
    　　3.安全工程师 Security Engineer
    　　移动互联网深入到生活方方面面的今天，网络安全在个人信息保护、隐私保护、企业数据保密方面显得尤为重要。
    　　安全工程师应具备这些技能――防火墙，Linux，网络安全，信息安全。市场对本职位的需求较去年上升了15%，帮助公司保护数字财产的专业人士的需求正在稳步增长，尤其是近年来数据安全事件频频发生，企业级数据安全开始受到了广泛重视。
    　　4.IoT架构师 loT Architect
    　　Randstad发现，物联网架构师最相关的技能是：Java，物联网，用户体验设计，云计算和AWS。
    　　然而，这并不足以成为IoT架构师。自由技术家Rick Delgado认为，物联网架构师应该培养一套有用的技能，这可以分为三个方面：技术，编程语言和个人技术路线规划。在一年内，该工作的市场需求上涨了41%，且2016年比2015年就上涨了113%。
    　　5.VR/AR工程师 VR/AR Engineer
    　　VR/AR工程师应具备以下技能：Java，C / C ++，C#，Python和虚拟现实。
    　　VR工程师目前的需求很高――这个职位的市场需求在短短一年内就上升了45%。由于是新兴行业，所以有很多创业公司涌现，市场的人才缺口一直很大，感兴趣的程序员可以考虑该方向。
    　　以后的互联网行业会越来越好，也会有越来越多的人加入到IT行业，高薪是一部分，还有一部分原因就是改变人们的生活，社会的进步离不开互联网，互联网一直在把生活变得越来越便捷化，个人觉得除了辛苦真的是很不错的行业。
    ?后记：对于大部分转行的人来说，找机会把自己的基础知识补齐，边工作边补基础知识，真心很重要。
